# Приключения Колобка
Игрок бросает вызов подземелью. 

Цель — пройти через 8 этажей возрастающей сложности, победить врагов и добраться до самого дна, где ждёт источник всего зла.


Игра построена на пошаговой механике: игрок делает ход (перемещение, атака, использование предмета), затем ходят враги. Подземелье представляет собой сетку тайлов, где каждая клетка может быть:



* Стеной — непроходимое препятствие

* Полом — свободная для передвижения область

* Лестницей — переход на следующий этаж

Карта генерируется программно: создаются случайные комнаты разного размера, которые соединяются коридорами. Это обеспечивает уникальность каждого прохождения.



### Видимость

Игрок видит только те области, которые находятся в радиусе обзора персонажа (8 клеток). Ранее посещённые места остаются видимыми на карте, но отображаются затемнёнными. Враги и предметы видны только в зоне прямой видимости, что добавляет элемент неожиданности и тактического планирования.



### Атаки

Бой происходит автоматически при попытке войти на клетку с врагом. Урон рассчитывается по формуле:



Урон = Атака атакующего - Защита цели + случайный модификатор (-2 до +2)

Существует шанс критического удара (15%), который удваивает нанесённый урон. После победы над врагом игрок получает очки опыта и с определённой вероятностью — золото.



### Магия

Персонаж может использовать заклинания, расходуя ману:


* Исцеление (20 маны) — восстанавливает здоровье в зависимости от уровня магии

* Огненный шар (30 маны) — наносит урон ближайшему видимому врагу

Мана восстанавливается при переходе на новый этаж или использовании зелий.



На каждом пятом этаже появляется босс — усиленный противник с увеличенными характеристиками и особым внешним видом. Победа над боссом приносит значительное количество опыта и золота.



Враги преследуют игрока, если видят его, и атакуют при сближении. Иногда они двигаются хаотично, что делает их поведение менее предсказуемым.



### Управление


`W` / ↑	Движение вверх

`S `/ ↓	Движение вниз

`A` / ←	Движение влево

`D` / →	Движение вправо



Enter	Спуститься по лестнице

< `I` >    Открыть инвентарь

< `1` >	    Заклинание исцеления

< `2` >	    Заклинание огненного шара

<`Esc`> 	Пауза / Выход в меню


### Звуковое сопровождение

Игра поддерживает фоновую музыку:



kolobok.mp3 — музыка в главном меню

game.mp3 — музыка во время игры

Музыка загружается из папки static/sounds и воспроизводится в цикле.
